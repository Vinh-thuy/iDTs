Target Organization & Staffing

Unified and Synergistic Team Structure

  • AI Core Function: engages with stakeholders, collects business needs, and explores solutions via a dedicated prototyping platform (e.g. Domino Data Lab). It performs short-cycle experimentation with inputs from Discovery, ServiceNow, Dynatrace, Insight Booster, etc., and regularly shares results in demos to break silos.
  • AI Function Exposition: ensures robust industrialization by integrating validated components into production platforms like Eureka.
  • Both teams interact continuously — not via traditional handovers — to ensure agility and delivery alignment.
Workflow Between AI Core Function and AI Function Exposition
AI Core Function
  • Identify AI Eureka features to implement
  • Collect needs from stakeholders
  • Prototype, run demos and validate concepts in Domino Data Lab
  • Develop internal AI tools (e.g., CodeHub for AI coding support)
→
AI Function Exposition
  • Manage the Knowledge Graph in production
  • Industrialize validated components
  • Evolve and extend Eureka features independently (e.g., integration into Teams, UI/UX development)
  • Ensure production quality
  • Deliver to users
Shared Data: Discovery, Dynatrace, ServiceNow, Insight Booster, Open Data — used by Core for prototyping, by Exposition for reliable production.
This organization maximizes speed, clarity, and collaboration — turning prototyped ideas into operational solutions without breaking continuity.
